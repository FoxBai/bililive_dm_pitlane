Pitlane Danmaku macOS

1. Drag PitlaneDanmaku.app to Applications.
2. Launch the app and fill in a Bilibili room ID or room URL.
3. Add this OBS browser source:
   http://127.0.0.1:17333/overlay
4. Keep the browser source background transparent.

Diagnostics:
- App settings and exported logs live in:
  ~/Library/Application Support/PitlaneDanmaku
- macOS crash reports live in:
  ~/Library/Logs/DiagnosticReports

If macOS blocks the app, use the signed and notarized DMG for public distribution.
